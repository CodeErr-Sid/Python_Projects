welcome!

This is a simple etch a sketch game using python turtle module,
what it does is it simply draws lines according to your command.


command:
  press key - 'w' to move forward
  press key - 's' to move backward
  press key - 'a' to move / point in left direction
  press key - 'd' to move / point in right direction

Thats it you are ready to go!...
